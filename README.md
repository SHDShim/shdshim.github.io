# Shim Research Group Website

This repository now contains a clean static GitHub Pages starter site for the
research group homepage at `https://shdshim.github.io/`.

## Current structure

- `index.html`: main homepage
- `styles.css`: site styling
- `.nojekyll`: tells GitHub Pages to serve files as plain static content

## Preserved old site

The previous site history was preserved locally before the reset:

- `archive-dev-2026-03-28`
- `archive-master-2026-03-28`

Those branches keep the earlier Pelican and published-site versions intact.

## Publish

Push the `master` branch to GitHub:

```bash
git push origin master
```

If GitHub Pages is configured to publish from `master` root, the site will
update automatically.
